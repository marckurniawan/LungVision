# 🫁 LungVision

**AI-powered Lung Cancer Risk Prediction using Machine Learning**

LungVision is a web-based application that predicts lung cancer risk using Machine Learning. The application analyzes demographic information, environmental and lifestyle factors, smoking history, and clinical symptoms to assist in early risk screening.

> **Disclaimer:** This application is intended for educational and research purposes only and is **not** a substitute for professional medical diagnosis.

---

## 🎯 Project Objectives

The main objectives of this project are to:

- Predict lung cancer risk using patient clinical data.
- Assist early screening through machine learning.
- Compare multiple machine learning models.
- Provide an interactive web application for prediction.

---

## 📊 Dataset

The dataset contains **1,000 patient records** with **25 clinical features**, including:

- Age
- Gender
- Air Pollution
- Alcohol Use
- Dust Allergy
- Occupational Hazards
- Genetic Risk
- Chronic Lung Disease
- Balanced Diet
- Obesity
- Smoking
- Passive Smoker
- Chest Pain
- Coughing of Blood
- Fatigue
- Weight Loss
- Shortness of Breath
- Wheezing
- Swallowing Difficulty
- Clubbing of Finger Nails
- Frequent Cold
- Dry Cough
- Snoring

**Target**

- Lung Cancer Risk
- Low / Medium / High (dataset)
- Binary prediction (Positive / Negative) for the trained model

---

## 🤖 Machine Learning Models

Several machine learning algorithms were evaluated:

- Logistic Regression ✅ (Selected Model)
- Random Forest
- XGBoost

The final deployed model is **Logistic Regression**, selected based on its superior recall performance for medical screening tasks.

---

## 📈 Model Performance

| Metric | Logistic Regression |
|---------|--------------------:|
| Accuracy | 61.0% |
| Precision | 61.6% |
| Recall | **95.9%** |
| F1 Score | **75.0%** |
| ROC AUC | 0.4751 |

High recall is prioritized because missing a positive lung cancer case (False Negative) is more critical than generating additional False Positives.
