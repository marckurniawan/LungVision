# 🫁 Lung Cancer Detection using Machine Learning

Project ini dibuat untuk membantu mendeteksi kemungkinan kanker paru-paru secara lebih awal menggunakan pendekatan **Machine Learning (Random Forest Classifier)**.

Ide dasarnya sederhana:  
dari data kesehatan pasien, model mencoba mengenali pola yang mungkin mengarah ke risiko kanker paru-paru.

---

## 🎯 Tujuan Project

Project ini bukan untuk menggantikan dokter, tapi untuk membantu sebagai **alat analisis awal (early screening)**.

Beberapa tujuan utamanya:
- Memprediksi risiko kanker paru-paru berdasarkan data pasien
- Membantu proses analisis data medis menjadi lebih cepat
- Menemukan pola yang mungkin tidak terlihat secara langsung

---

## 📊 Dataset yang Digunakan

Dataset berisi informasi kebiasaan dan kondisi kesehatan pasien, seperti:

- Usia (Age)
- Kebiasaan merokok (Smoking)
- Yellow Fingers
- Anxiety
- Peer Pressure
- Penyakit kronis (Chronic Disease)
- Kelelahan (Fatigue)
- Wheezing
- Konsumsi alkohol
- Batuk (Coughing)
- Sesak napas (Shortness of Breath)

🎯 Target yang diprediksi:
- `Lung Cancer: YES / NO`

---

## 🛠️ Tech Stack

Project ini dibuat menggunakan:

- Python 3.13+
- Pandas & NumPy (pengolahan data)
- Matplotlib & Seaborn (visualisasi)
- Scikit-learn (machine learning)
- Jupyter Notebook (.ipynb)

---

## 🧠 Model yang Digunakan

Model utama yang digunakan adalah:

### 🌲 Random Forest Classifier

Kenapa Random Forest?
- Cukup akurat untuk data tabular
- Stabil meskipun data memiliki noise
- Bisa menangani banyak fitur sekaligus
- Cocok untuk eksperimen awal di data medis
